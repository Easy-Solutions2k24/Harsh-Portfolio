Easy ZR

A multi feature GUI application

Description

Easy-ZR is modern multi-featured desktop application developed with the aim to provide multiple file handling mechanisms into one, whether its archiving, extracting, compression, decompression, Easy-ZRCD is there at your service.

Easy-ZR is consists of the following features into one desktop app:-

  1. Archiver - .zip, .rar
  2. Extractor

All the above functionalities works in sync to provide an extensible area to explore around with.

Easy-ZR is python-java based, with the rich set of python modules induced in the application development process, Easy-ZRCD utilizes multiple python modules. Being a Tkinter GUI based application, it is easy to use, develop, and is cross platform with portable compact code design.
